Знакомство с классом Container. В Bootstrap много различных классов, с помощью которых можно делать разметку. Используем класс container.

 

Контейнеры - это фундаментальный строительный блок Bootstrap, который содержит, дополняет и выравнивает содержание в пределах заданного устройства или области просмотра.

 

Контейнеры являются основным элементом макета в Bootstrap и необходимы при использовании системы сеток по умолчанию. Также они используются для хранения, заполнения и (иногда) центрирования содержимого внутри них. Хотя контейнеры могут быть вложенными, для большинства макетов вложенный контейнер не требуется. В зависимости от класса контейнера разметка будет увеличиваться или уменьшаться.

Для этого на сайте «getbootstrap.com» в разделе «Docs» перейдите в «Containers» (рис. 3.16).

Как работают колонки? Они основаны на архитектуре «flexbox сетки». Flexbox означает, что у нас есть опции для изменения отдельных колонок и изменения групп столбцов на уровне строки.

При создании макетов сетки вся информация размещается в колонках. Иерархия сетки Bootstrap строится следующим образом:

1.       Контейнер.

2.       Строка.

3.       Колонка.

4.       Содержание.

Далее обращаемся к тегу <а>, применяем цвет и отбрасывание тени. Обновляем страницу на localhost. Появилось выделение ссылок с небольшим закруглением (рис. 3.32).

 

Text-decoration в позиции none отключает подчеркивание на странице.

Border-radius – создает закругления у углов текста на 10 пикселей. Чем большее значение пикселей будет указано, тем больше будут закругления.

Color: mediumslateblue – применяем цвет, как у заголовка.

Box-shadow: inset – отбрасываемая тень блока с внешней стороны на внутреннюю.

Первое 0 – указывает на отсутствие теней.

Второе - 3px – значение смещения тени на -3 пикселя (значение размытия).

Третье 0 – значение свойства по умолчанию для нашей тени (значение растяжения).

Текст

*{

    font-family: 'Cavet', cursive;

}

html,

h1,

h2,

h3,

h4,

h5,

h6{

    color: mediumslateblue;

}

 

body{

    background-color: bisque;

 
 
}

a{

    text-decoration: none;

    border-radius: 10px;

    color: mediumslateblue;

    box-shadow: inset 0 -3px 0 mediumslateblue;

}

Контейнеры - это фундаментальный строительный блок Bootstrap, который содержит, дополняет и выравнивает содержание в пределах заданного устройства или области просмотра.

Свойство display - отвечает за вывод и визуальное отображение элементов на странице.

Свойство padding – края отступа.

Элемент «float»– элемент, который отображается на нашем контексте.
