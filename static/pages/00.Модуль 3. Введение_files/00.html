Описание программы

Программа "Разработчик Junior: программирование бизнес-приложений, игр и сайтов. Изучаемые языки: 1С, JavaScript, HTML, CSS, PHP, MySQL, C#" состоит из 4 модулей по 36 ак. часов. Каждый модуль программы является логическим продолжением предыдущего, рекомендуем проходить их последовательно друг за другом:

1 МОДУЛЬ. "МИНИ-ИГРЫ 1С"

Первый модуль посвящен созданию мини-игр на языке "1С:Предприятие 8". Изучая основы программирования, ученики научились создавать 2D-игры, создали игру-квест с собственным сюжетом и научились создавать карточные игры. На примере создания игральных кубиков освоили начальную логику и синтаксис программирования.

2 МОДУЛЬ. "1С:МАГНАТ"

Во втором модуле ученикам предстоит создать свой первый бизнес-проект. После создания нескольких игр появляется необходимость реализации более серьезной игры. Теперь с помощью платформы "1С:Предприятие 8" создадим игру с торгово-учетной системой. Задача игрока в новой разработке – не уйти в ноль и выполнять заказы клиентов. В игровом формате вместе создадим собственное приложение для оптимизации бизнес-процесса.

3 МОДУЛЬ. "СОЗДАНИЕ СОЦИАЛЬНОЙ СЕТИ С НУЛЯ"

В третьем модуле ученики разработают сайт с системой аутентификации пользователя с последующей его авторизацией. Мы познакомимся с такими языками программирования, как HTML, CSS, JavaScript, узнаем, как работает база данных и создадим собственный веб-дизайн.

4 МОДУЛЬ. РАЗРАБОТКА 3D-ИГРЫ  

В четвертом модуле ученики познакомятся с одним из самых популярных движков для создания видеоигр – Unity.  Мы познакомимся с синтаксисом и интерфейсом, рассмотрим применение физики и анимации для создания собственного бесконечного раннера с подсчетом очков и записью рекордов игроков.

Введение  

Добро пожаловать на третий модуль программы "Разработчик Junior: программирование бизнес-приложений, игр и сайтов. Изучаемые языки: 1С, JavaScript, HTML, CSS, PHP, MySQL, C#". В рамках третьего модуля мы создадим сайт, через который пользователи смогут обмениваться своими размышлениями на интересующие их темы, писать статьи и комментировать работы других пользователей. У каждого будет свой отдельный аккаунт, вход в который будет осуществляться с помощью логина и пароля.

Благодаря этому модулю, дети получат навыки разработки frontend, backend, которые дадут базовые знания программирования на разных языках, что поможет для развертки своего собственного сайта в будущем при работе в компании.

Для начала обучения на этом модуле каждый ученик должен завершить прохождение модулей 1-2, на которых:

·         изучались принципы разработки и использования конфигураций на базе платформы "1С:Предприятие 8.3";

·         формировалось представление об особенностях программирования на встроенном языке;

·         проходило знакомство со внутренними механизмами написания кода на платформе "1С:Предприятие 8.3"

 

Если 2 модуля пройдены, можно начать обучение!

На примере разработки этого сайта мы изучим основы веб-дизайна, взаимодействия между базой данных и серверной частью. В результате у каждого получится свой индивидуальный проект с персональным дизайном и доработками.
