Прикрепление картинок. Оптимизация ошибок

Следующим этапом, для которого нужна была связь, станет подвязка изображения. Необходимо написать код, позволяющий добавлять изображение в базу данных названием. Добавим строку в path.php, где realpath() указывает путь к домену «socialsite»:

<?php

 

define("BASE_URL", ‘http: //localhost/Socialsite/");

 

define("ROOT_PATH", realpath((dirname(_FILE_))));

 

 

Функция «realpath()» - раскрывает все символические ссылки, переходы типа /./, /../ и лишние символы / в пути path, возвращая канонизированный абсолютный путь к файлу.

Загрузка одного файла - чтобы форма отправила файл, необходимо использовать только метод POST для отправки данных и добавить к тегу <form> атрибут enctype="multipart/form-data", который определяет способ кодирования данных формы при их отправке на сервер.

Загрузка нескольких файлов - для загрузки сразу нескольких файлов к <input type="file"> нужно добавить атрибут multiple, а к имени поля – [].
