Далее заходим в папку «application» и создаем там еще одну под названием «database». Переходим в нее и создаем файл «connect.php», который как раз будет выполнять функцию подключения к базе данных.

<?php

$db_name = ‘socialsite’; #проверяем название папки

$db_user = ‘root’;       #указываем имя пользователя

$db_pass = ‘mysql’;      #указываем пароль

$charset = ‘utf8;        #указываем кодировку

$options =[PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION]; #устанавливаем подключение PDO с атрибутом ATTR_ERRMODE в ERRMODE_EXCEPTION

 

#с помощью метода try подключаемся в БД

try{

 

    $pdo = new PDO("mysql:host=localhost;dbname=$db_name;charset=$charset", $db_user, $db_pass, $options);

}catch(PDOException $i){

    die("Ошибка подключения к базе") #в случае ошибки отрабатывается метод PDO

}

 

Переходим обратно в проект по адресу «localhost/socialsite/application/database/connect.php». Если при переходе не возникло ошибки, то подключение к базе данных прошло успешно.
